# Testing Scripts

This directory contains scripts for testing various functionality.

## Scripts

- `test_admin_login.py` - Test admin authentication
- `debug_login.py` - Debug login issues
- `test_delete.py` - Test delete functionality
- `add_test_data.py` - Add sample data for testing

## Usage

Run tests from the project root directory:
```bash
python scripts/testing/script_name.py
```

These scripts help verify that features are working correctly during development.
